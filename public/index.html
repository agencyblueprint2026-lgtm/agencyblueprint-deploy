01 — The Story

I'm not a developer.
I'm a craftsman.

I'm a realism portrait artist and product maker. I built a brand, a website, an e-commerce system, and a physical product line — essentially solo, from scratch.

At some point I started delegating the repetitive work to AI agents. Not because I understood the technology deeply — but because I understood what I needed done, and I could think structurally about how to describe it.

That turned out to be enough.

This guide exists because most resources assume you know Python. They assume you can debug JSON errors. They assume you're comfortable with infrastructure.

Most people aren't. I wasn't. And I built working agent systems anyway.

The Agency Blueprint is what I wish had existed when I started.

"Automation is not about replacing yourself. It is about delegating structured tasks so you can focus on judgment, creativity, and strategy."
— The Agency Blueprint, Preface
02 — The System

Five working agents.
One clear progression.

01
The Echo Bot
Understand the fundamental Input → Process → Output loop. Every agent you ever build starts here.
02
The Web Researcher
Give your agent eyes. Connect it to live data. Learn how to filter signal from noise before it reaches the LLM.
03
The Inbox Manager
Introduce decision routing. Your agent reads, classifies, and acts differently based on context. This is foundational agency behaviour.
04
The Content Factory
Build a multi-stage production pipeline. Sequential dependency. Variable passing. Real output ready for publishing.
05
The Orchestra
Three specialised agents working in parallel, synthesised into one coherent output. Division of cognitive labour.
+
Advanced Foundations
Prompt architecture. Memory and retrieval. Security. Failure modes. Cost management. 25 business agent ideas to build next.
